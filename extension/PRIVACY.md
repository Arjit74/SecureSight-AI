# Privacy Policy - GuardianLink

## Data Collection
URLs scanned are analyzed using GuardianLink's proprietary security models:
- Advanced Signature Analysis Engine
- Reputation Analysis Models
- Safe Browsing Security Models
  
- Local logs stored in browser storage only
- No personal data collected
- Analysis performed locally and on our secure backend

## Data Retention
- Scan logs deleted when extension uninstalled
- Users can manually clear logs via dashboard

## Security Analysis Services
All security analysis is performed using GuardianLink's proprietary machine learning models and heuristic engines - no third-party threat intelligence APIs are used.
